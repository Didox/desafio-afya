## Project setup

### Clone this repository:

```
git clone git@github.com:jsenger/desafio-afya.git
```

### Enter frontend directory:

```
cd frontend
```

### Install dependencies:

```
yarn
```

### Compile and hot-reload for development:

```
yarn start
```

### Compile and minify for production:

```
yarn build
```
